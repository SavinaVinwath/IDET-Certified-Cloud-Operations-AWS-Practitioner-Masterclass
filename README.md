# IDET Certified Cloud Operations AWS Practitioner Masterclass Notes Repository

This repository collects the day-by-day notes and demo materials for the IDET Certified Cloud Operations AWS Practitioner Masterclass.

## Course Overview

Each day focuses on a different AWS topic, from the early class notes in Days 1 to 3 through hands-on labs for networking, compute, storage, databases, serverless, billing, security, Terraform, Docker, and ECS.

## Quick Use

Open the folder for the day you want to study, then read the matching markdown file or demo assets inside that folder.

## Cloning Instructions

1. Clone the repository, replacing <repository_url> with the actual URL.

   git clone <repository_url>

2. Change into the project directory.

   cd IDET-Certified-Cloud-Operations-AWS-Practitioner-Masterclass

3. Browse the materials for the relevant day, for example.

   cd "Day 1"

If you prefer SSH, use:

git clone git@github.com:<username>/<repo>.git

If you need the repository URL or further assistance, open an issue or contact the maintainer.
